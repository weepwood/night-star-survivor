# Night Star Survivor

一款以《吸血鬼幸存者》式自动战斗为基础、采用暗夜像素幻想视觉风格的纯前端网页游戏。

## 当前内容

- 三名属性与初始能力不同的守夜者
- 星矢阵列、月轮刃、寒星领域、星雷链四种武器
- 遗物组合与武器进化
- 普通、精英、远程和首领敌人
- 阶段切换、首领警报和地面灾害
- 冲刺、星爆主动技能
- 像素粒子、硬边拖尾、位图伤害数字和整数屏幕震动
- 基于 Web Audio 的轻量合成音效

## Pixel-perfect 渲染

- 战场逻辑分辨率：`640 × 360`
- 世界地图尺寸：`2560 × 1440`
- 镜头使用整数坐标和移动死区跟随玩家
- 所有主要角色、敌人、弹道和粒子均按整数像素绘制
- Canvas 禁用图像平滑，显示层使用最近邻缩放
- 缩放倍率结合 `devicePixelRatio` 计算，使每个逻辑像素尽量对应整数个物理像素
- 桌面端通常显示为 `1280 × 720`，高分屏可使用更高整数倍率
- 可通过页面顶部的“物理像素”按钮手动切换 1×、2×、3×、4×或自动模式

## 操作

- `WASD` / 方向键：移动
- `Space`：冲刺
- `Q`：星爆
- `P` / `Esc`：暂停

## 本地运行

这是一个纯静态项目，可直接打开 `index.html`，也可以使用任意静态服务器：

```bash
python -m http.server 8080
```

随后访问 `http://localhost:8080`。

## 部署

仓库包含 GitHub Pages 工作流。修改合并到 `main` 后，会自动发布静态站点。
