# Night Star Survivor

一款以《吸血鬼幸存者》式自动战斗为基础、采用暗夜像素幻想视觉风格的纯前端网页游戏。

## 当前内容

- 三名属性与初始能力不同的守夜者
- 星矢阵列、月轮刃、寒星领域、星雷链四种武器
- 遗物组合与武器进化
- 普通、精英、远程和首领敌人
- 阶段切换、首领警报和地面灾害
- 冲刺、星爆主动技能
- 粒子、动态光照、拖尾、伤害数字、残影和屏幕震动
- 基于 Web Audio 的轻量合成音效
- 移动端自适应界面

## 操作

- `WASD` / 方向键：移动
- `Space`：冲刺
- `Q`：星爆
- `P` / `Esc`：暂停

## 本地运行

这是一个纯静态项目，可直接打开 `index.html`，也可以使用任意静态服务器：

```bash
python -m http.server 8080
```

随后访问 `http://localhost:8080`。

## 部署

仓库包含 GitHub Pages 工作流。修改合并到 `main` 后，会自动发布静态站点。
